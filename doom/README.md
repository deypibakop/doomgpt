# Doom — Roasting AI agent

This folder contains a small, self-contained "Doom" agent that generates playful roasts.

Files:
- doom.py — a simple Python CLI that prints roasts.
- DOOM_PROMPT.txt — example system prompt for an LLM-based Doom.
- README.md — instructions and safety notes.

Safety: The agent includes a simple guardrail to avoid roasting protected classes or obvious personal identifiers. Always use responsibly.
